NIST CSS-less Header Template

This repo can be used as a source for the NIST headers (and soon: footers and exit scripts) that are required(*) for use on public-facing sites. It is recommended that you DO NOT copy these files into your repo, but rather link to them in-place so that you will automatically receive any changes or updates.

An example of a minimal header bar is here.

An example of a full header is here.

NOTE: this process is still being developed (improved, simplified). Check back often for updates until we get this capability fully implemented.

To use this repo you do NOT have to copy any files; just insert the following into your pages' HEAD section: 


<link rel="stylesheet" href="https://pages.nist.gov/nist-header-footer/css/nist_fonts.css">
<link rel="stylesheet" href="https://pages.nist.gov/nist-header-footer/css/nist_header.css">
<script src="https://pages.nist.gov/nist-header-footer/js/jquery-1.9.0.min.js" type="text/javascript" defer="defer"></script>
<script src="https://pages.nist.gov/nist-header-footer/js/nist-header.js" type="text/javascript" defer="defer"></script>
